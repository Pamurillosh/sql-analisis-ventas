#Análisis de Ventas con SQL

##Objetivo
Analizar el comportamiento de ventas para identificar productos más vendidos y clientes más importantes.

##Estructura
Se crearon dos tablas:
- clientes
- ventas

##Técnicas utilizadas
- JOIN
- GROUP BY
- HAVING
- Funciones agregadas (SUM)

##Resultados
- Identificación del producto más vendido
- Análisis de ventas por cliente
- Segmentación de clientes con mayor gasto

##Conclusión
El análisis permite identificar patrones de consumo y clientes clave, facilitando la toma de decisiones comerciales.

##Herramientas
- SQL Server
